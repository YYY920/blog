## Local Development

Requirements:
- [Node.js](https://nodejs.org/): 18.0.0+
- [Bun](https://bun.sh/) (recommended) or npm

Clone the repository:

```shell
git clone https://github.com/jennyyang920/blog.git
cd blog
```

Common commands:

```shell
# Install dependencies
bun install

# Start development server
bun dev

# Build project
bun run build

# Preview (after build)
bun preview
```
